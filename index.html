Documentacion de meteor JS

Meteor, o MeteorJS, es un framework para aplicaciones web con JavaScript libre y de código abierto[3]​ escrito usando Node.js. Meteor facilita un la creación rápida de prototipos y produce código multiplataforma (web, Android, iOS). Se integra con MongoDB y usa Distributed Data Protocol y un patrón publish-subscribe para propagar automáticamente al cliente cambios en los datos sin requerir que el desarrollador escriba algún código de sincronización. En el cliente, Meteor depende de jQuery y puede ser usado con cualquier librería de UI para JavaScript.

Instalando meteorjs en Windows 10

  • Meteor supports Windows 7/Windows Server 2008 R2 and up.
  • The installer uses Chocolatey, which has its own requirements.
  • Disabling antivirus (Windows Defender, etc.) will improve performance.
  • iOS development is not supported.
